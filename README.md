# articles-hub

这个仓库用于沉淀 AI 学习笔记、实践代码与 Go 相关示例。

## 仓库结构

- `ai-基础知识/`：从零构建理解 Agent 系列内容（含章节文章与配套代码）
- `ai-思考/`：对 Agent、反馈系统等主题的延展思考
- `ai-tools/`：工具实践记录
- `logfacade/`：一个可复用的 Go 日志封装示例
- `assets/`：仓库配图等静态资源
